# Blood Bank Management System

This project is developed using HTML, CSS, JavaScript, and PHP. 
It manages donor registration, blood stock, requests, and hospital records efficiently.

Features:
- Donor Registration
- Blood Availability Search
- Stock Management
- Admin Panel
